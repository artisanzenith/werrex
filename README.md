# Nexus Digital — Agency Website

A modern, production-ready digital agency website built with React, Tailwind CSS, and Framer Motion.

## Features

- Dark premium UI with gradients and animations
- Three services with interactive pricing modals (Basic, Standard, Premium)
- Full sections: Hero, About, Services, Portfolio, Testimonials, Team, FAQ, Contact, Newsletter
- Responsive design (mobile, tablet, desktop)
- SEO meta tags and favicon
- Loading screen with smooth transitions

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18
- Vite 6
- Tailwind CSS 3
- Framer Motion
- React Router
- React Helmet Async
